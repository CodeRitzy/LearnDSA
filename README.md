# LearnDSA
LearnDSA is a full-stack educational platform designed to help users master data structures used in coding interviews and systems programming. The platform combines structured lessons, active recall, quizzes, spaced repetition, and progress tracking so users can retain concepts instead of forgetting them after one pass.

# Features
- Structured lessons for core data structures
- Active recall prompts after each topic
- Quizzes for concept reinforcement
- Spaced repetition review scheduling
- Progress dashboard with mastery tracking
- Coding exercises tied to each structure
- Systems-oriented explanations and tradeoffs

# Built with
Next.js, TypeScript, Tailwind CSS, PostgreSQL, Prisma, Clerk/Auth.js

# Goal
The goal of LearnDSA is to make data structure learning more effective, more interactive, and more retention-focused than a typical tutorial website.
